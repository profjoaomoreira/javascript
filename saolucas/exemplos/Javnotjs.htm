Eu já mencionei isso antes, mas acho que é importante repetir: Java não é JavaScript! Há muitas diferenças. Muitas das mensagens postadas sobre Java no newsgroup comp.lang.javascript provam que existe um tremenda falta de informação. E do lado contrário, as questões sobre JavaScript deverão ser postadas em comp.lang.javascript, e  NÃO em comp.lang.java. As técnicas usadas em Java e em JavaScript diferem tanto porque o Java é uma linguagem de programação, e o JavaScript é uma linguagem de scripting (roteiros). São duas linguagens diferentes com propósitos diferentes. Eu devo admitir, é claro, que esses dois nomes quase similares parecem se referir à mesma coisa; só que isso não é verdade! Leve isto em conta, por favor, quando você fizer uma pergunta a alguém sobre Java ou JavaScript.
